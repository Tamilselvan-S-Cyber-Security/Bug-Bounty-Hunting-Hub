Essential Recon Tools

Five must-have tools for efficient bug bounty reconnaissance and data processing.

1. anew

Purpose: Adds new lines to a file, skipping duplicates (great for recon pipelines).

Commands

cat new.txt | anew all.txt
subfinder -d target.com | anew subs.txt

2. gf (grep on steroids)

Purpose: Wrapper around grep to filter URLs/parameters using predefined patterns (xss, sqli, redirect, etc.).

Commands

cat urls.txt | gf xss
cat urls.txt | gf redirect
cat urls.txt | gf sqli

3. uro

Purpose: Declutters URLs by removing duplicates, tracking params, and useless variations.

Commands

cat urls.txt | uro > cleanurls.txt
gau target.com | uro

4. unfurl

Purpose: Extracts parts of URLs (params, keys, values, paths, domains).

Commands

cat urls.txt | unfurl keys
cat urls.txt | unfurl values
cat urls.txt | unfurl paths
cat urls.txt | unfurl domains

5. qsreplace

Purpose: Replaces all query string values with a supplied value (FUZZ, payloads, etc.).

Commands

cat urls.txt | qsreplace FUZZ
cat urls.txt | qsreplace '"><script>alert(1)</script>'

Quick Summary

Tool Purpose
anew Add only new results
gf Filter URLs by vuln type
uro Clean & normalize URLs
unfurl Extract URL components
qsreplace Replace parameter values
