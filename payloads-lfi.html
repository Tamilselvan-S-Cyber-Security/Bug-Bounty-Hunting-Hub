LFI Payloads

Local File Inclusion payload collection for testing.

Folder Contents

This folder contains Local File Inclusion payloads for testing file access vulnerabilities.

Linux File Targets

File Description
/etc/passwd User account information
/etc/shadow Password hashes (requires root)
/etc/hosts Network host entries
/etc/hostname System hostname
/proc/self/environ Environment variables
/proc/self/cmdline Process command line
/proc/self/status Process status info
/var/log/apache2/access.log Apache access logs
/var/log/apache2/error.log Apache error logs
/var/www/html/index.php Web root files

Windows File Targets

File Description
C:\Windows\win.ini Windows configuration
C:\Windows\System32\drivers\etc\hosts Hosts file
C:\inetpub\wwwroot\web.config IIS configuration
C:\Windows\System32\config\SAM Password database
