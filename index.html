Reconnaissance

Passive and active subdomain enumeration techniques

Passive Subdomain Enumeration

  • subfinder -d target.com -all -recursive -silent
  • amass enum -passive -d target.com
  • assetfinder --subs-only target.com
  • findomain -t target.com -q

Active Subdomain Discovery

  • amass enum -active -d target.com
  • gobuster dns -d target.com -w wordlist.txt
  • puredns bruteforce wordlist.txt target.com
  • massdns -r resolvers.txt -t A subdomains.txt

Probing & Deep Recon

  • httpx -l resolved_subs.txt -p 80,443,8080
  • katana -list live_websites.txt -jc
  • gau --subs target.com | anew urls.txt
  • paramspider --domain target.com

JavaScript Analysis

  • katana -list live_websites.txt -jc | grep \"\.js$\"
  • cat js_all.txt | mantra
  • python3 SecretFinder.py -i js_files.txt
  • cat js_files.txt | gf api-keys
